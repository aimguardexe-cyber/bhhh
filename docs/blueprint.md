# **App Name**: Coderyn Dashboard

## Core Features:

- Dashboard Overview: Provides a visual overview of key software platform metrics, including total products, library items, and plan status.
- Product Management: Enables users to add, view, and manage products within the software platform.
- Library Access: Allows users to access and manage a library of resources and documentation.
- User Profile Management: Facilitates management of user profiles, including updating personal information and account settings.
- Responsive Layout: Adapts the dashboard layout to different screen sizes for optimal viewing on desktops, tablets, and mobile devices.

## Style Guidelines:

- Primary color: Soft beige (#EAE4D5) to establish a warm and inviting tone for the dashboard.
- Background color: Light gray (#F2F2F2), providing a neutral backdrop that is easy on the eyes.
- Accent color: Muted olive (#B6B09F) for interactive elements and key highlights.
- Body and headline font: 'Poppins', a geometric sans-serif for a precise and contemporary look. Note: currently only Google Fonts are supported.
- Use simple, minimalist icons from a set like 'Feather' to maintain a clean and modern aesthetic.
- Implement a sidebar navigation with collapsible functionality for smaller screens.
- Design a card-based layout for the main dashboard content to improve visual organization.
